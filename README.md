# CoreKit CollectionView

CollectionViews with ease!



## Installation



### CocoaPods

```
source 'https://github.com/CoreKit/CocoaPods.git'
pod 'CollectionView', '~> 2.0.0'
```



### Carthage

```
github "CoreKit/CollectionView" "2.0.0"
```
